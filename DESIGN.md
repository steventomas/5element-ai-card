# DESIGN.md

## 气质与意象
"第五元素"取自科幻经典，意指超越水火土风之外的至高力量——AI正是企业的"第五元素"。视觉锚点：深空观测舱，全息投影界面漂浮于漆黑虚空，能量流线贯穿，偶尔闪过琥珀暖光代表人文温度。整体气质：克制、精密、有力量感。

## 视觉策略
- 深空黑底 + 全息光效，光源自上而下微倾
- 图形语言：几何线条、六边形网格、能量流线、光点粒子
- 卡片采用毛玻璃（glassmorphism）质感，边缘微光描边

## 配色方案
- 主背景：#0a0e1a（深空黑）
- 渐变起始：#0f172a → #020617
- 主强调色：#00d4ff（全息青），意象来源：全息投影的标志性色
- 辅助强调：#7c3aed（能量紫），意象来源：宇宙射线
- 暖点缀：#f59e0b（琥珀金），意象来源：数据节点闪烁的微光
- 文字主色：#f0f4ff（冷白）
- 文字次色：#94a3b8（银河灰）
- 卡片背景：rgba(15, 23, 42, 0.6) + backdrop-blur

## 字体排版
- 标题字体：思源黑体 / Noto Sans SC（Google Fonts CN 域）
- 正文字体：系统默认无衬线栈
- 标题加粗 700，正文 400
- 排版节奏：标题大字距、正文紧凑，层级间用间距而非字号拉开

## 动效与交互
- 页面加载：品牌名从模糊到清晰（0.6s ease-out）
- 滚动渐入：元素从下方 20px 淡入（0.5s ease-out，IntersectionObserver 触发）
- 按钮：按下 scale(0.96) + 亮度微增，松开回弹
- 光效：背景有缓慢流动的渐变光斑（8s 循环），卡片 hover 边缘发光增强
- 数字/图标：微浮动动画（3s ease-in-out infinite）

## 设计禁忌
- 禁止使用蓝色纯色块 + 白字的"IT公司标配"风格
- 禁止圆角超过 20px 的圆润卡通卡片
- 禁止彩虹渐变或超过 3 色的渐变
- 禁止弹窗/模态遮挡主要内容
- 禁止花哨粒子系统导致页面卡顿（仅用 CSS 实现）
